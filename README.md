# Online Food Delivery System using SQL

* Project Overview
This project is an advanced SQL-based Online Food Delivery Database Management System designed to manage customers, restaurants, menu items, orders, and delivery details efficiently.

** Features
- Customer Management
- Restaurant Management
- Order Tracking
- Delivery Tracking
- Revenue Analysis
- Food Order Analytics

** SQL Concepts Used
- Joins
- Aggregate Functions
- GROUP BY & HAVING
- Subqueries
- CTE (Common Table Expression)
- Views
- Stored Procedures
- Triggers
- CASE Statements
- Window Functions

** Tables Used
- Customers
- Restaurants
- Menu
- Orders
- Delivery

** Tools Used
- MySQL
- SQL Workbench
- GitHub

** Advanced Features
- Revenue Analysis
- Customer Spending Analysis
- Restaurant Ranking
- Automated Trigger Operations
- Delivery Status Updates

** Outcome
This project helped in understanding real-world database management and advanced SQL concepts used in food delivery platforms.
